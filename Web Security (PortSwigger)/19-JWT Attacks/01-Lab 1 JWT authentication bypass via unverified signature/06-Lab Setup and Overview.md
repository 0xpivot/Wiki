## Lab Setup and Overview

### Accessing the Lab

To access the lab, follow these steps:

1. Visit the URL `https://portswigger.net/web-security`.
2. Click on the "Sign Up" button to create an account if you don't already have one.
3. Once logged in, navigate to the "Academy" section.
4. Select "All Content" and then "All Labs".
5. Search for "JOT labs" and select "Lab Number One: JOT Authentication Bypass via Unverified Signature".

### Lab Objective

The objective of this lab is to bypass authentication by modifying the session token to gain access to the admin panel at `/admin` and delete the user "Carlos". You will be provided with the credentials of a regular user account to start the lab.
