## Real-World Examples and Recent Breaches

### Real-World Example: CVE-2021-21972

CVE-2021-21972 is a DOM-based XSS vulnerability found in the popular web conferencing platform Zoom. The vulnerability allowed attackers to inject malicious scripts into the DOM of the Zoom client, potentially leading to unauthorized access and data theft.

### Real-World Example: CVE-2020-14182

CVE-2020-14182 is another DOM-based XSS vulnerability found in the web-based email client Roundcube. The vulnerability allowed attackers to inject malicious scripts into the DOM of the Roundcube client, potentially leading to unauthorized access and data theft.

### Real-World Example: CVE-2019-11324

CVE-2019-11324 is a DOM-based XSS vulnerability found in the web-based file manager FileZilla. The vulnerability allowed attackers to inject malicious scripts into the DOM of the FileZilla client, potentially leading to unauthorized access and data theft.
