## Lab Setup and Environment

To understand and practice DOM-based XSS vulnerabilities, we will use the Web Security Academy provided by PortSwigger. This environment allows us to explore various web security vulnerabilities in a controlled and educational manner.

### Accessing the Lab

1. **Sign Up**: Visit the URL `https://portswigger.net/web-security` and click on the sign-up button to create an account.
2. **Login**: Once you have an account, log in to the Web Security Academy.
3. **Navigate to Labs**: Click on the "Academy" tab and select "All Labs".
4. **Search for DOM-Based Vulnerabilities**: Use the search bar to find the lab titled "DOMXSS using web messages and JSON.parse".

### Lab Overview

The lab we will be working on involves exploiting a DOM-based XSS vulnerability using web messages and JSON parsing. The goal is to construct an HTML page on the exploit server that exploits this vulnerability and calls the `print` function in the victim's browser.
