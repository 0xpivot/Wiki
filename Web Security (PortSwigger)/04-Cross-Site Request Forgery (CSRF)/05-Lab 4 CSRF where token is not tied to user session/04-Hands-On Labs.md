## Hands-On Labs

To practice testing and defending against CSRF attacks, consider the following hands-on labs:

### PortSwigger Web Security Academy

PortSwigger Web Security Academy offers a comprehensive lab on CSRF attacks. You can practice identifying and exploiting CSRF vulnerabilities in a controlled environment.

### OWASP Juice Shop

OWASP Juice Shop is a deliberately insecure web application that includes various security vulnerabilities, including CSRF. You can practice identifying and exploiting CSRF vulnerabilities in a real-world application.

### DVWA (Damn Vulnerable Web Application)

DVWA is a PHP/MySQL web application that contains numerous security vulnerabilities, including CSRF. You can practice identifying and exploiting CSRF vulnerabilities in a controlled environment.

### WebGoat

WebGoat is an interactive, gamified training application that teaches web security concepts, including CSRF. You can practice identifying and exploiting CSRF vulnerabilities in a controlled environment.

By practicing in these labs, you can gain hands-on experience in identifying and defending against CSRF attacks.
