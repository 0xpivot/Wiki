## Lab Setup

To access the lab, follow these steps:

1. Visit the URL: `https://portswigger.net/web-security`.
2. Click on the "Sign Up" button to create an account.
3. Once logged in, navigate to the "Academy" section.
4. Search for "cross-site scripting labs".
5. Select lab number nine titled "Reflected XSS into a JavaScript string with angle brackets HTML encoded".

### Accessing the Lab Environment

The lab environment is set up using Burp Suite, which is a popular tool for web application security testing. All your requests are intercepted by Burp Proxy, allowing you to inspect and modify them.
