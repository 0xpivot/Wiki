## Real-World Examples

### Recent CVEs and Breaches

#### CVE-2019-11324: AngularJS DOM-Based XSS

In 2019, a DOM-based XSS vulnerability was discovered in AngularJS versions 1.x. The vulnerability allowed attackers to inject malicious scripts into the DOM through user input. This CVE highlights the importance of proper input validation and output encoding in web applications.

#### Real-World Breach: Capital One Data Breach

In 2019, Capital One suffered a significant data breach where an attacker exploited a misconfigured web application firewall (WAF) to access sensitive customer data. While this breach was primarily due to a misconfiguration, it underscores the broader risks associated with web application vulnerabilities, including XSS.
