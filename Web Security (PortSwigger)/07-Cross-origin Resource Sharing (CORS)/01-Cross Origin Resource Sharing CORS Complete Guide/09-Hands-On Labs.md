## Hands-On Labs

To practice and reinforce your understanding of Same Origin Policy and CORS, consider the following hands-on labs:

- **PortSwigger Web Security Academy:** Offers interactive labs to practice web security concepts, including Same Origin Policy and CORS.
- **OWASP Juice Shop:** A deliberately insecure web application for practicing web security skills.
- **DVWA (Damn Vulnerable Web Application):** A PHP/MySQL web application that is vulnerable to many types of web application attacks.

By working through these labs, you can gain practical experience in identifying and mitigating Same Origin Policy and CORS vulnerabilities.
