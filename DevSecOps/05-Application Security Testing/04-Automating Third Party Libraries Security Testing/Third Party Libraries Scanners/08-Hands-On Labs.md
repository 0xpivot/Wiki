## Hands-On Labs

### PortSwigger Web Security Academy

PortSwigger Web Security Academy offers a comprehensive set of labs for learning about web application security, including third-party library security testing. These labs cover various aspects of security testing and provide practical experience in identifying and mitigating vulnerabilities.

### OWASP Juice Shop

OWASP Juice Shop is an intentionally insecure web application designed for security training. It includes numerous vulnerabilities, including those related to third-party libraries, providing a realistic environment for practicing security testing.

### DVWA (Damn Vulnerable Web Application)

DVWA is another intentionally insecure web application designed for security training. It includes various vulnerabilities, including those related to third-party libraries, providing a realistic environment for practicing security testing.

### WebGoat

WebGoat is an interactive training application designed to teach web application security. It includes numerous lessons on identifying and mitigating vulnerabilities, including those related to third-party libraries.
