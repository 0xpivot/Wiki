## Hands-On Labs

### PortSwigger Web Security Academy

PortSwigger Web Security Academy is a comprehensive online platform that provides hands-on labs for web application security. The platform covers various topics, including SQL injection, cross-site scripting (XSS), and broken authentication.

#### Example Lab: SQL Injection

In this lab, learners will practice identifying and exploiting SQL injection vulnerabilities in a web application. The lab provides a simulated web application and guides learners through the process of identifying and exploiting the vulnerability.

### OWASP Juice Shop

OWASP Juice Shop is an intentionally vulnerable web application designed for security training. The application contains numerous security vulnerabilities, including SQL injection, XSS, and broken authentication.

#### Example Lab: Broken Authentication

In this lab, learners will practice identifying and exploiting broken authentication vulnerabilities in the OWASP Juice Shop. The lab provides a simulated web application and guides learners through the process of identifying and exploiting the vulnerability.

### DVWA (Damn Vulnerable Web Application)

DVWA is a PHP/MySQL web application that is intentionally vulnerable to common web application vulnerabilities. The application is designed for security training and provides a wide range of vulnerabilities for learners to practice identifying and exploiting.

#### Example Lab: Command Injection

In this lab, learners will practice identifying and exploiting command injection vulnerabilities in the DVWA. The lab provides a simulated web application and guides learners through the process of identifying and exploiting the vulnerability.
