## Hands-On Labs

### Recommended Labs

For hands-on practice with integrating a CI/CD pipeline with AWS ECR, consider the following labs:

1. **PortSwigger Web Security Academy**
   - Focuses on web application security but includes modules on CI/CD pipelines and container security.
   - URL: [PortSwigger Web Security Academy](https://portswigger.net/web-security-academy)

2. **OWASP Juice Shop**
   - A deliberately insecure web application for security training.
   - URL: [OWASP Juice Shop](https://owasp.org/www-project-juice-shop/)

3. **CloudGoat**
   - Provides hands-on experience with AWS security and compliance.
   - URL: [CloudGoat](https://cloudgoat.readthedocs.io/en/latest/)

4. **flaws.cloud**
   - Offers a variety of cloud security labs, including CI/CD pipeline security.
   - URL: [flaws.cloud](https://flaws.cloud/)

These labs will help you gain practical experience in setting up and securing CI/CD pipelines with AWS ECR.

By following these detailed explanations and examples, you should have a comprehensive understanding of how to integrate a CI/CD pipeline with AWS ECR, manage environment variables, authenticate with ECR, tag and push images, and handle expired tokens.
