## Hands-On Labs

### PortSwigger Web Security Academy

PortSwigger Web Security Academy is a popular hands-on lab for learning web application security. It provides a series of interactive challenges and exercises that cover various security topics, including SQL injection, cross-site scripting (XSS), and broken authentication.

### OWASP Juice Shop

OWASP Juice Shop is another popular hands-on lab for learning web application security. It provides a vulnerable web application that simulates a modern e-commerce platform. Users can explore and exploit various security vulnerabilities, including SQL injection, XSS, and insecure direct object references.

### DVWA (Damn Vulnerable Web Application)

DVWA is a PHP/MySQL web application that is intentionally designed to be vulnerable. It provides a series of security challenges that cover various topics, including SQL injection, XSS, and command injection. Users can learn how to exploit these vulnerabilities and how to defend against them.

### WebGoat

WebGoat is a hands-on lab for learning web application security. It provides a series of interactive lessons and exercises that cover various security topics, including SQL injection, XSS, and broken authentication. Users can learn how to exploit these vulnerabilities and how to defend against them.
