## Hands-On Labs

To gain practical experience with GitOps and ArgoCD, consider the following hands-on labs:

- **PortSwigger Web Security Academy**: Focuses on web application security but also covers CI/CD pipelines and GitOps principles.
- **OWASP Juice Shop**: A deliberately insecure web application for practicing web security skills, including CI/CD pipelines.
- **Kubernetes Goat**: A hands-on lab for learning Kubernetes security, including GitOps and ArgoCD.

These labs provide a safe environment to practice and experiment with GitOps and ArgoCD, helping you to master the concepts and techniques covered in this chapter.
