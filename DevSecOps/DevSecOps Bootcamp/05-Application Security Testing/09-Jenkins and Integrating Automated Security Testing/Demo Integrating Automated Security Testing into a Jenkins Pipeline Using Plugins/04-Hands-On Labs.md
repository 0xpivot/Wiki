## Hands-On Labs

### Recommended Labs

- **PortSwigger Web Security Academy**: Offers practical exercises on web security.
- **OWASP Juice Shop**: Provides a vulnerable web application for security testing.
- **DVWA**: A deliberately insecure web application for practicing security testing.

### Example Lab Exercise

1. **Setup Jenkins**: Install and configure Jenkins.
2. **Install Dependency Track**: Set up Dependency Track and configure it in Jenkins.
3. **Create Jenkinsfile**: Write a Jenkinsfile to include security testing.
4. **Run Pipeline**: Execute the pipeline and verify the results.
