## Practice Questions & Answers

**Q1. How do you print a number in Python?**

To print a number in Python, you use the `print()` function followed by the number inside the parentheses. For example, to print the number 1, you would write:

```python
print(1)
```

When you run this code, it will display the number `1` in the console.

**Q2. Can you print a string in Python? How?**

Yes, you can print a string in Python using the `print()` function. You need to enclose the string within quotes (either single `'` or double `"`). Here’s an example:

```python
print("Hello, world!")
```

This will display the text `Hello, world!` in the console when executed.

**Q3. What happens if you try to print multiple items in Python?**

If you try to print multiple items in Python, you can separate them by commas inside the `print()` function. The items will be printed in sequence, separated by spaces by default. For example:

```python
print(1, 2, 3)
```

This will display `1 2 3` in the console.

**Q4. How can you control the separator between items when printing multiple items in Python?**

To control the separator between items when printing multiple items, you can use the `sep` parameter in the `print()` function. By default, the separator is a space, but you can change it to any character or string. For example:

```python
print(1, 2, 3, sep='-')
```

This will display `1-2-3` in the console, with dashes as separators.

**Q5. How do you print a variable in Python?**

To print a variable in Python, you simply include the variable name inside the `print()` function. For example, if you have a variable `x` set to `10`, you would write:

```python
x = 10
print(x)
```

This will display `10` in the console.

**Q6. How can you print formatted strings in Python?**

In Python, you can print formatted strings using f-strings (formatted string literals), which are prefixed with `f` and allow you to embed expressions inside string literals. For example:

```python
name = "Alice"
age = 30
print(f"My name is {name} and I am {age} years old.")
```

This will display `My name is Alice and I am 30 years old.` in the console.
