## Real-World Examples and Recent Breaches

### Example: AWS S3 Bucket Exposure

In 2020, a major breach occurred where sensitive data was exposed due to misconfigured S3 buckets. This highlights the importance of proper configuration and management of infrastructure resources.

### Example: Terraform State File Exposure

In 2021, a company experienced a breach where their Terraform state file was exposed, leading to unauthorized access to their infrastructure. This underscores the importance of securing sensitive information and using proper access controls.

### How to Prevent / Defend

#### Detection

Regularly scan for misconfigurations and vulnerabilities using tools like AWS Security Hub or Terraform Sentinel.

#### Prevention

1. **Use Proper Access Controls**: Ensure that proper access controls are in place to prevent unauthorized access.
2. **Encrypt Sensitive Data**: Encrypt sensitive data both at rest and in transit.
3. **Regular Audits**: Perform regular audits of infrastructure configurations to identify and remediate vulnerabilities.
